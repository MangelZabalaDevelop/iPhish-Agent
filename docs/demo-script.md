# Demo Script

Target length: 2 to 3 minutes.

## Shot 1: Problem

Explain that phishing-awareness campaigns are important but operationally
complex. A consultant usually has to coordinate copy, landing pages, SMTP,
recipients, review, screenshots, and approvals manually.

## Shot 2: Local Stack

Show NVIDIA AI Workbench with the iPhish Agent project. Point out the four apps:
Iphish, GoPhish, Mailpit, and ComfyUI.

Message: this is cloneable Workbench packaging on Dell Pro Max with GB10.

## Shot 3: Agent

Open Iphish. Show Hermes connected to the local model alias and tools.

Message: the agent does not need a cloud model; it uses a local
OpenAI-compatible endpoint.

## Shot 4: Campaign Creation

Ask the agent to create an authorized awareness campaign. Show the agent using
the controlled GoPhish skill and safe review flow.

Message: this is offensive-security knowledge used defensively.

## Shot 5: Review Gate

Open Mailpit and show the review email. Open the landing page through the
Workbench GoPhish route.

Message: nothing goes to real SMTP until a human approves the reviewed content.

## Shot 6: Safety And Business Value

Summarize:

- Local model and local campaign data.
- OpenShell guardrail model.
- No passwords, MFA codes, tokens, payment data, or secrets.
- Faster campaigns for corporate security-awareness programs.
- Compact Dell Pro Max with GB10 as the local AI workstation.
